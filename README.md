# Zomato District: Product Portfolio

A focused product management portfolio exploring the **District** vertical by Zomato. This repository contains strategic case studies, product requirement documents (PRDs), and go-to-market (GTM) plans aimed at enhancing the "Going Out" experience.

## 🚀 Projects Included

### 1. Ratings & Reviews for Amusements
A comprehensive review system designed to build trust within the District app for activities like go-karting, amusement parks, and sports.
*   **Artifact:** [Reviews PRD](./District_Activities_Reviews_PRD_Naveen_Gupta.pdf)

### 2. Plan with Friends (Group Planning)
A social feature allowing users to coordinate outings, split payments, and invite friends directly within the Zomato ecosystem.
*   **Artifact:** [Group Planning PRD](./District_PlanWithFriends_PRD_Naveen_Gupta.pdf)

### 3. GTM Strategy: Activities Vertical
A detailed city-by-city rollout plan for District's amusement vertical, backed by Q3 FY25 financial data.
*   **Artifact:** [GTM Plan](./District_Activities_GTM_Naveen_Gupta.pdf)

### 4. District Play Pass
An opportunity brief exploring a subscription-based model for fitness and sports courts (similar to FitPass but integrated into District).
*   **Artifact:** [Opportunity Brief](./District_PlayPass_OpportunityBrief_Naveen_Gupta.pdf)
